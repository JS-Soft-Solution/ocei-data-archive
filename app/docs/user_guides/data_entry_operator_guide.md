# Data Entry Operator User Guide
## Legacy Permit Digitization System

---

## Overview

As a **Data Entry Operator**, you are responsible for digitizing legacy permit records from paper registers into the system. This guide will help you complete applications for Ex-Electrician, Ex-Supervisor, and Ex-Contractor permits.

---

## Getting Started

### Accessing Your Dashboard

1. Log in to the system with your credentials
2. Navigate to the **Legacy Permits** section from the main menu
3. Select the permit type you want to work on:
   - Ex-Electrician
   - Ex-Supervisor
   - Ex-Contractor

## Creating a New Application

### Method 1: Create from Scratch

1. Click **"New Application"** button
2. Fill in the old certificate number
3. Complete all required fields across 5 tabs
4. Upload required attachments
5. Submit for approval

### Method 2: Claim Existing Record

If a record already exists in the system but hasn't been claimed:

1. Scroll to **"Claim Existing Record"** section
2. Enter the **Old Certificate Number**
3. Click **"Claim Record"**
4. System will assign the record to you
5. Complete the application and submit

## Saving Your Work

### Save Draft
- Click **"Save Draft"** at any time
- Your progress is saved
- You can come back later to complete
- Status remains "Draft"

### Save & Next
- Completes current tab
- Automatically moves to next tab
- Fast way to progress through the form

### Final Submission

**Requirements before submitting:**
1. All 5 tabs completed
2. At least 1 attachment uploaded
3. No validation errors

**To Submit:**
1. Go to Tab 5
2. Verify all information
3. Click **"Submit for Approval"**
4. Confirm submission in popup
5. Application moves to Office Assistant queue

---

## Handling Rejections

### If Office Assistant Rejects

1. You'll see status change to **"Rejected by OA"**
2. Click **"Reason"** button to see rejection details
3. Click **"Edit"** to make corrections
4. Fix the issues mentioned
5. Click **"Submit"** again
   - Goes back to Office Assistant

### If Secretary Rejects

1. Status changes to **"Rejected by Secretary"**
2. View detailed rejection reason
3. Make significant corrections
4. Click **"Submit"** 
   - **Important:** Goes directly back to Secretary (skips OA)

## Common Mistakes to Avoid

‚ùå **Don't:**
- Submit without attachments
- Use duplicate old certificate numbers
- Leave required fields (*) empty
- Upload files larger than 10MB
- Upload unsupported file types (only PDF, JPG, PNG allowed)

‚úÖ **Do:**
- Double-check certificate numbers
- Verify NID numbers
- Upload clear, readable documents
- Save frequently
- Read rejection reasons carefully

---

## Tips for Efficiency

1. **Use Save & Next** to move quickly through tabs
2. **Upload attachments early** so you don't forget
3. **Use Bulk Submit** for multiple completed applications
4. **Claim records** if you have the certificate number
5. **Check status regularly** to handle rejections promptly
